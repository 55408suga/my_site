# Kazushi Suga's Portfolio

## 概要
このウェブサイトは、学校の講座でGemini CLIを用いて作成されたポートフォリオサイトです。自己紹介、スキル、趣味（ボルダリングジム情報を含む）をまとめています。

## 使用技術
-   HTML5
-   CSS3
-   JavaScript (matrix.js for background animation)
-   Gemini CLI (サイト作成支援)

## 機能一覧
-   自己紹介セクション
-   スキルリスト (Node.js, Python/Flask) と対応するアイコン
-   趣味（ボルダリング）の紹介
-   横浜市内のボルダリングジム一覧とその公式サイトへのリンク (クリック可能なブロック形式)
-   応答性の高い2カラムレイアウト (サイドバーとメインコンテンツ)
-   連絡先情報 (GitHub)
-   動的な背景アニメーション (Matrix効果)
